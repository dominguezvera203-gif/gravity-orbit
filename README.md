(ITMSD 3) Laboratory 1: One Life + One Button

GravityOrbitGame

A simple React Native + Expo mobile game created for the Mobile Application Development – Game Jam Lab. The game follows the One Button + One Life rule where the player must rely on quick reactions to survive.

Mobile App Game Jam – Laboratory 1

Course: Mobile Application Development Framework: React Native + Expo
Concept: One Button + One Life Game

Game Overview

GravityOrbitGame is a simple reaction-based game where the player controls a planet orbiting around a moving sun. The goal is to survive as long as possible while collecting stars.
The player uses only one button to control the orbit of the planet. Stars appear randomly and collecting them activates power-ups such as speed, invincible, and bonus. Each power-up is activated after collecting three stars of the same type.
If the planet touches the sun without an active power-up, the game immediately ends because the player only has one life. The game focuses on timing, focus, and quick reactions while keeping the gameplay simple.

Reflection

What was your game idea?

My game idea was to create a simple one-button game called Gravity Orbit where the player controls a planet orbiting around a sun. The goal is to survive while collecting stars that activate power-ups. I wanted the game to be easy to understand but still fun and challenging.

What was the most difficult part to implement?

The most difficult part was managing the game state and collisions between the planet, sun, and stars. At first, the invincible power-up did not work correctly because the planet would still die when touching the sun. I had to adjust the collision logic to check if the power-up was active.
Another challenge was making the stars spawn randomly without appearing too often.

What would you improve with more time?

If I had more time, I would improve the animations and visual effects to make the game more engaging. I would also add better feedback when collecting stars and possibly add a score multiplier for collecting stars consecutively.
